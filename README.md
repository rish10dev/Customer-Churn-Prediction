# Customer Churn Prediction

## Overview
This project predicts customer churn using Machine Learning techniques on the Telco Customer Churn dataset.

## Features
- Data Cleaning and Preprocessing
- Exploratory Data Analysis
- Churn Distribution Visualization
- Logistic Regression Model
- Model Saving using Joblib

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-Learn
- Joblib

## Dataset
Telco Customer Churn Dataset
